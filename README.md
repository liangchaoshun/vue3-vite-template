### 推荐 IDE: webstorm

### 为了与平台同步，建议使用 npm

### webstorm 效率快捷键

- 从磁盘重新索引文件：`Ctrl + Alt + Y`
- 优化 import 导入（如，剔除 unused module）：`Ctrl + Alt + O`
- 用 `prettier`格式化文件：`Ctrl + Shift + Alt + P`
- 显示组件关系图：`Ctrl + Alt + U`
